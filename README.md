## ğŸŒ Deployment

This project is deployed via **GitHub Pages**.

To deploy:

1. Ensure your `package.json` includes:

   ```json
   "homepage": "https://jderenthalcs.github.io/JustinDerenthal.dev",
   "scripts": {
     "predeploy": "npm run build",
     "deploy": "gh-pages -d build"
   }
   ```
2. Deploy:

   ```bash
   npm run deploy
   ```
